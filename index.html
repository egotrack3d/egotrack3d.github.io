Hololens Object Tracking Dataset

Collecting and Benchmarking ego-centric object tracking-style data

Link to the Dataset

This site hosts a 3d object tracking dataset focused on working with egocentric data. We utilize the Microsoft Hololens2 and its vast sensor suite to collect 12 different scenes. In addition, we will provide 3 types of annotations to work with. The first annotation per-frame 3d object location. The second annotation is an object's binary state of static/dynamic where "static" pertains to an object not moving and a "dynamic" state referring to an object currently being interacted by the camera wearer. The third annotation is the per-frame bounding box in each sequence.

EgoTrack3D Available Data

Sensor Suite

  • 1 RGB Camera (1280x720) @ 30 fps
  • 4 VLC Grayscale Cameras (640x480) @ 30 fps
  • 1 ToF Camera Ahat (512x512) @ 45 fps and Longthrow (320x288) @ 5 fps

Additional Estimated Information

  • Base Map
  • Camera Pose
  • Sensor Intrinsics/Extrinsics

Scene Information

For each scene we collect 5 regular sequences and 1 SLAM sequence. All sensors are collected to use its most recent information relative to the RGB camera. Each sequence is around 5 minutes each

Regular Sequence

  • Base map (.obj)
  • All sensor poses (.json)
  • Sensor data (.png)
  • ToF uses Ahat (512x512) @ 45 fps

SLAM Sequence

  • Base map (.obj)
  • All sensor poses (.json)
  • Sensor data (.png)
  • ToF uses Longthrow (320x288) @ 5 fps
